Class Stack

A Stack is a tool for managing DOM elements as groups. Within a group, dom elements can be added, removed, moved around. The group can be moved to another parent node while keeping the DOM elements in the same order, excluding the parent dom elements's children that are not in the Stack.
Defined in: Stack.js.
